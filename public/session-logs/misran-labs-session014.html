01

Contexte & différenciateur

EXP-005 est la plus spectaculaire visuellement. Là où EXP-001 et EXP-002 outputent du texte formaté, EXP-005 génère du HTML rendu dans un iframe — le visiteur voit une vraie landing page apparaître en temps réel après le streaming du code.

⚡ Décision clé — style SaaS moderne clair (pas Misran Labs) : les landing pages générées utilisent un style blanc/gris clair, typographie Inter, gradients colorés. Le contraste dark portfolio / light SaaS dans l'iframe renforce l'illusion de génération externe. Un recruteur perçoit immédiatement que "quelque chose a été créé".

Flow utilisateur
  • Chips cliquables → remplissent le textarea
  • Clic GÉNÉRER → streaming du code HTML en direct
  • Toggle CODE / PREVIEW → iframe avec vraie page rendue
  • Meta affiché : nom produit, tokens simulés, temps
  • Bouton NOUVELLE GÉNÉRATION → reset complet
4 cas pré-définis
  • TaskFlow — gestion tâches remote (violet #7c3aed)
  • BudgetAI — suivi budget personnel (vert #059669)
  • FeedbackLoop — feedback client automatisé (orange #ea580c)
  • LaunchKit — fallback générique (bleu #2563eb)
02

Architecture technique

Même pattern de simulation que EXP-001 et EXP-002 — zéro appel API réel, zéro surcoût sur l'abonnement Pro. Le streaming est un setInterval qui avance de 8 caractères toutes les 15ms sur le HTML hardcodé.

États React
  • phase : 'idle' | 'streaming' | 'done'
  • view : 'code' | 'preview'
  • streamIndex : avance de 8 chars / 15ms
  • currentData : { code, meta } sélectionné
  • input : valeur du textarea
Colonne droite — OUTPUT
  • Phase idle : message d'attente centré
  • View code : terminal #060a10, JetBrains Mono, curseur ▋
  • View preview : iframe srcdoc={currentData.code}
  • Toggle CODE / PREVIEW désactivé si idle
  • Bouton actif mis en évidence (border teal)
03

Fix post-déploiement — chips UI

Même problème qu'EXP-001 à la première génération : les chips de suggestion ressemblaient à des champs de formulaire en colonne. Corrigé immédiatement après screenshot de validation.

Fix 01
Chips en colonne → horizontaux
flex-direction column → flex-wrap wrap. Pattern chip tag reconnaissable immédiatement.
Fix 02
Style neutre → teal design system
color #4a7a94 → #25e2cc, border opaque → rgba teal. Cohérence avec EXP-001 et le reste du portfolio.

⚠ Pattern récurrent identifié : les chips générés par Claude Code tendent à être stylisés comme des inputs en colonne. À préciser systématiquement dans les prochains prompts : "chips horizontaux flex-wrap, style teal, pas des inputs".
