Comment fonctionne la facturation Claude Pro

L'abonnement Claude Pro ($20/mois) et l'API Claude sont deux produits distincts avec deux facturations distinctes. Claude Code utilise le quota de l'abonnement — pas l'API. L'extra usage est la passerelle entre les deux.

Abonnement Pro — $20/mois
  • Quota hebdomadaire de tokens inclus
  • Couvre claude.ai + Claude Code
  • Quota atteint → bloqué jusqu'au renouvellement
  • Renouvellement automatique chaque semaine
  • Zéro facturation supplémentaire si pas d'extra usage
Extra Usage — pay-per-use
  • Activable dans Settings → Usage
  • Facturé aux tarifs API standard (aucune majoration)
  • Nécessite une carte bancaire + prépaiement
  • Cap mensuel configurable pour éviter les surprises
  • Apparaît comme ligne séparée sur la facture

03

Human-directed vs Agents autonomes

Si le même travail avait été délégué à des agents autonomes plutôt qu'orchestré manuellement, le coût aurait été radicalement différent. Voilà pourquoi.

Pourquoi les agents coûtent plus cher

Un agent fait ce que Claude Code fait, mais en boucle autonome. À chaque itération il relit tout le contexte, réfléchit à la stratégie, génère, vérifie, corrige, recommence. Chaque tour de boucle = une nouvelle requête API complète.

1 Human qui orchestre — par tâche
  • Toi qui lis visuellement → 0 token
  • Toi qui décides → 0 token
  • Prompt ciblé ~500 tokens input
  • Lecture fichiers concernés ~2000 tokens
  • Génération code ~3000 tokens output
  • Total : ~5 500 tokens / tâche
Agent autonome — même tâche
  • Lecture projet entier ~15 000 tokens
  • Réflexion stratégique ~3 000 tokens
  • Génération ~3 000 tokens
  • Vérification automatique ~5 000 tokens
  • Correction si erreur → nouveau tour complet
  • Total : ~30 000+ tokens / tâche

💡 Insight clé : être dans la boucle n'est pas un frein — c'est un avantage. Tu lis visuellement, tu décides, tu formules un prompt ciblé. Ces trois actions humaines coûtent zéro token et remplacent des dizaines de milliers de tokens de vérification automatique. Le "1 Human · 100% AI-assisted" du lab est la configuration optimale sur tous les axes : coût, qualité, contrôle, apprentissage.
